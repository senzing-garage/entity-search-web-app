EDA Tools Console

As of version 2.0, Senzing ships with 3 python scripts that help you explore your data: G2Explorer.py, G2Snapshot.py, and G2Audit.py. These were formerly known as the POC (Proof of Concept) utilities in prior Senzing versions. We renamed them to EDA (Exploratory Data Analysis) tools because while they certainly help POCs they can be used long after to help ensure your system is performing as expected. Click here for more information on what each tool does and a brief walkthrough of functionality.
